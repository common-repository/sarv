== Description ==

Sarv Email Marketing platform allows you to experience how easy it's to create beautiful and personalized email campaigns. Now you can send a right email to the right people at the right time. You'll get automation, drag n drop editor, advanced segmentation, A/B or split testing, advanced reporting, and more tools in one place. 


== SETTING UP Sarv Email Marketing ==

Sarv Subscription And Email Marketing Plugin those will provide free signup account and create list and form and then apply on WordPress website where you can manage form as we as drag and drop form fields.

The form will provide shortcode those can use on any place in WordPress website.

[First, you’ll want to set up a Sarv account. (It’s free.)](https://sarv.com/email-marketing/pricing/cost-per-subscribers/)

Then, install the Sarv plugin and select Sarv from the Settings menu in your WordPress sidebar.

== Installation ==

1. Upload the `sarv` folder to the `/wp-content/plugins/` directory
2. Activate the **Sarv Email Marketing** plugin through the 'Plugins' menu in WordPress
3. Configure the plugin by going to the `Sarv` page that appears in your Settings menu


== Frequently Asked Questions ==

= How do I install the tracking script on my website? =

This plugin automatically installs. 
All you have to do is install this plugin and enter your Account ID on its settings page.
Your Account ID can be found on the [Settings > Site Setup](https://www.sarv.com/)
page.
